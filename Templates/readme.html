üöÄ API Endpoints

Get Shelters

  • Method: GET
  • Endpoint: /Shelters
  • Description: Retrieve the list of shelters.

Add Shelter

  • Method: POST
  • Endpoint: /ShelterCreation
  • Description: Add a new shelter.

Update Shelter

  • Method: PUT
  • Endpoint: /ShelterUp
  • Description: Update shelter information.

Delete Shelter

  • Method: DELETE
  • Endpoint: /ShelterDel/:id
  • Description: Delete a shelter.

üßæ Explanation

  • View the list of shelters:
      Shelters tab will send a request to the server via /Shelters endpoint where it will retreive the data that is found in data.JSON file.

  • Add new shelters:
      Adding new shelter is a form in the website that will send a POST request to the server via /CreateShelter endpoint where it will pass the parameters to it via JSON.

  • Update existing shelter information:
      When updating a shelter the website will send a PUT request to the server via /ShelterUpdate endpoint where it will pass the parameters to it via JSON.

  • Delete shelters:
      When deleting an existing shelter the website will send a DELETE request to the server via /ShelterDelete endpoint where it will pass the id to the server and perform the remove according to the id passed.
