🏠 Shelter Locator

📜 Description

Shelter Locator is a simple web application,

that allows users to locate, update, delete, and add new shelters. 🚀

🚀 API's for the project:

The Shelter Locator project exposes the following API endpoints:

  • `GET  /getShelters`: Retrieve the list of shelters.
  • `POST  /addShelter`: Add a new shelter.
  • `PUT  /updateShelter`: Update shelter information.
  • `DELETE  deleteShelter/:id`: Delete a shelter.

🛠 Adding a new shelter

To add a new shelter:

  • Make a `POST` request to the `/addShelters` endpoint.
  • Include the required information in the request body.

    • Or:

  • You can simply go to the add shelter page and add there.

🛠 Usage

Shelter Locator provides a user-friendly interface to:

  • View the list of shelters
  • Add new shelters
  • Update existing shelter information
  • Delete shelters
